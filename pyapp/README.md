# Thermal Analyzer Pro - Python Edition

This directory contains a simplified rewrite of the original web
application in pure Python using **Streamlit**.

## Features

- Upload thermal images or `.bmt` files
- Display the thermal image and optional RGB photo
- Interactive histogram of the thermal data using Plotly

## Usage

Install the required dependencies and run the app with Streamlit:

```bash
pip install -r requirements.txt
streamlit run main.py
```

The application will start locally at `http://localhost:8501`.
